# Pi练手项目-一本好书：书籍推荐详情页

#### Description
Pi练手项目-一本好书：基于原生三件套的书籍推荐详情页（静态页面），出于练习的目的引入了jQuery库，以及为了兼容各浏览器格式引入了day.js库
特点：
1.以js控制状态，插入css动画改善观感
2.使用localStorage模拟后台储存，让静态页面产生交互性
